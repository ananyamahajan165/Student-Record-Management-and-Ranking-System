# Student Record Management and Ranking System

## 📌 Project Description

Student Record Management and Ranking System is a web-based application developed using Node.js and Express. The system allows efficient management of student records including adding, viewing, deleting students and automatically generating rankings based on marks.

The project demonstrates client-server architecture, RESTful API development, routing, file handling, and JSON-based data storage.

---

## 🚀 Features

- Add Student Records
- View All Students
- Delete Student by Roll Number
- Automatic Ranking Based on Marks
- JSON File Storage
- REST API Testing using Thunder Client
- Clean and Modern UI

---

## 🛠 Technologies Used

- Node.js
- Express.js
- JavaScript
- HTML (Hyper Text Markup Language)
- CSS
- Thunder Client (API Testing)
